, 7 = Lainnya), status perkawinan (Kawin, Belum kawin), pekerjaan (dengan kode 1 = PNS, 2 = TNI/POLRI, 3 = Pedagang, 4 = Petani, 5 = Nelayan, 6 = Buruh)
